# 项目背景：shadow's homepage

这是一个个人仪表板/主页，包含天气、搜索、倒计时和统计功能。使用原生 JavaScript、CSS 和 HTML 构建。

## 工程标准

- **原生优先：** 坚持使用原生 JavaScript (ES6+) 和 CSS。除非明确要求，否则避免添加沉重的框架。
- **项目结构：**
    - 逻辑代码位于 `js/`。
    - 样式表位于 `css/`。
    - 静态数据位于 `data/data.js`。
    - 第三方库存储在 `vendor/`。
- **命名规范：** CSS 类名使用 `kebab-case`，JavaScript 变量和函数名使用 `camelCase`。
- **性能：** 保持页面轻量且加载迅速。

## 交互偏好

- **直接行动：** 对于小的 UI 调整或逻辑修复，直接实施。
- **文档维护：** 增加重要功能时，请同步更新 `CHANGELOG.md`。
- **语言：** 所有交流和文档说明均使用中文。
